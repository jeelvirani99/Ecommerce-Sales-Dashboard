#  HR Analytics Dashboard

Professional HR analytics tool for analyzing employee performance, satisfaction, and attrition patterns using advanced data visualization.

##  Features

- **Attrition Analysis**: Identify departments with high turnover
- **Performance Tracking**: Monitor employee performance scores
- **Salary Insights**: Analyze compensation across departments
- **Satisfaction Metrics**: Track employee satisfaction levels
- **Correlation Analysis**: Find relationships between variables

##  Technologies

- Python 3.x
- Numpy-matrix operation
- Pandas - Data manipulation
- Matplotlib - Additional plotting

##  Usage
```bash
python Ecommerce-Sales-Dashboard.py
```
